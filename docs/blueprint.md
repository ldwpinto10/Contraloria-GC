# **App Name**: ArqueoPro

## Core Features:

- Petty Cash Setup: Define and manage the initial assigned value for each petty cash fund.
- Transaction Entry: Input cash debits and reimbursement invoice details, including amounts, descriptions, and categories.
- Real-time Balance Tracker: View the current petty cash balance, sum of transactions, and the variance against the initial fund value as items are entered.
- Transaction History: Access a historical list of all recorded petty cash transactions for a specified period, stored for auditing purposes.
- Reconciliation Report Generation: Generate a detailed reconciliation (cuadre) report, summarizing the initial fund, total expenses, cash balance, and any discrepancy for physical verification.
- AI-Powered Variance Insight: An AI tool to analyze the reconciliation report, identifying and highlighting significant discrepancies or patterns that require attention.

## Style Guidelines:

- Primary color: A stable and trustworthy deep blue (#2465D6) to signify precision and professionalism.
- Background color: A very light, desaturated blue-grey (#EEF2F7) to provide a clean and calm workspace, reducing eye strain.
- Accent color: A medium bright sky blue (#6DC6D6) for interactive elements and highlights, ensuring clear differentiation without being overwhelming.
- Headline and body font: 'Inter' (sans-serif) for its modern, objective, and highly legible appearance suitable for numerical data and reports.
- Use clear and concise line-art icons that functionally represent financial operations and data management, promoting intuitive navigation.
- Implement a structured, column-based layout with generous white space to enhance readability and organization of financial data and forms.
- Subtle, quick transition animations on form submissions and report generation to provide visual feedback and a responsive feel without distraction.
